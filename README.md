# SD-1
Lab Report 1. The topics covered in this lab report include:

Java Intro: Introduction to Java and creating the first Java program.
Java Get Started: Setting up a Java development environment and running a simple Java program.
Java Syntax: Understanding the basic syntax, including structure and conventions in Java.
Java User Input: Taking input from the user using the Scanner class.
Java Output: Displaying output to the console using System.out.println().
Java Comments: Writing single-line and multi-line comments to explain code.
Java Variables: Declaring and initializing variables in Java.
Java Data Types: Using different data types like int, double, char, boolean, etc.
Java Type Casting: Converting between different data types (implicit and explicit casting).
Java Operators: Using arithmetic, relational, and logical operators in Java.
Java Strings: Working with String manipulation methods.
Java Math: Using mathematical operations with the Math class (like Math.sqrt() and Math.pow()).
Java Booleans: Using boolean values for conditions and logic.
Java If...Else: Conditional statements to execute different blocks of code.
Java Switch: Using the switch statement for multiple conditions.
Java While Loop: Repeating code using a while loop.
Java For Loop: Iterating through a block of code using a for loop.
Java Break/Continue: Controlling loop execution using break and continue.
Java Arrays: Storing multiple values in an array and iterating over it.
